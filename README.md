# ReCiclo

## Aluno:
Francisco Vitoria Alves de Jesus

## Proposta de Valor

O aplicativo ReCiclo conecta pessoas que desejam doar objetos com pessoas que precisam deles, promovendo sustentabilidade, reutilização e impacto social positivo.

## Descrição das Telas

### 1. Home / Feed
Tela inicial com itens disponíveis próximos ao usuário.

### 2. Filtros
Permite buscar por distância, categoria e estado de conservação.

### 3. Detalhes do Item
Mostra foto, descrição e informações do doador.

### 4. Cadastro de Item
Usuário cadastra objetos para doação.

### 5. Chat
Comunicação entre doador e interessado.

### 6. Perfil / Conquistas
Exibe dados do usuário e selos conquistados.

## Jornada Principal

O usuário abre o app, procura itens disponíveis, utiliza filtros, acessa detalhes do produto, conversa pelo chat e conclui a retirada, recebendo um selo ecológico.
